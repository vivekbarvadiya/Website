LVC Instructions

Before the class :

  • Tech Check: Ensure you have a stable internet connection (check speed requirements if specified by the class provider), a working webcam, microphone, and headphones (optional but recommended).
  • Software familiarity: If you're using a specific platform, familiarize yourself with its interface beforehand. Many offer tutorials or guides

During the Class:

  • Join the Session: Log in to the virtual classroom a few minutes before the start time using the provided link or information.
  • Etiquette: Mute your microphone when not speaking to minimize background noise. Use the chat function for quick questions or raise your virtual hand to actively participate.
